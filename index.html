Quant Dashboard

투자 리서치 프로젝트 통합 대시보드

모멘텀 팩터, D램(DRAM) 가격, 베스트 팩터 결과를 한 화면에서 빠르게 훑어보고 각 프로젝트 원본으로 바로 이동하는 개인 리서치 허브입니다.

데이터 정책

이 페이지는 세 프로젝트의 공개 GitHub Pages JSON만 best-effort로 읽습니다. 원본 프로젝트 코드나 산출물을 수정하지 않으며, 데이터가 비어도 바로가기와 설명은 유지됩니다.

본 페이지는 개인 리서치 대시보드이며 투자, 세무, 법률 또는 매매 조언이 아닙니다. 원본 프로젝트의 공개 데이터가 지연되거나 변경될 수 있습니다.
